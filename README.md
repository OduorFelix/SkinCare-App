# SkinCare-App

This is a React Native (Cross-Platform) SkinCare Products App for a modern-day shops!


The Main Screens

![Screenshot](screenshot.png)


Tech Stack Used:
1. `Expo`
2. `Typescript`

To Run in your local environment:

1. Clone the repo
2. In your terminal, run `yarn install`
3. Run `yarn start`
4. Scan the QR Code with Expo App
